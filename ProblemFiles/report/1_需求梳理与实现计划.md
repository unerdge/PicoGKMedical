# 3D盒型画布设计器 - 完整需求梳理与实现计划

## 项目概述

**核心定位：** 预算约束下的最优解求解器（不是模板工具，不是专业CAD）
**目标用户：** 最终用户（品牌方/电商卖家/创业者），零包装知识要求
**核心价值：** 花最少的钱做到刚好够用

### 第二阶段 - 物理引擎（核心壁垒）

#### 5. 结构力学算法
- [ ] **McKee公式（BCT抗压强度）**
  ```
  BCT = 5.874 × ECT × CAL^0.508 × P^0.492
  ```
  - [ ] ECT值数据库（材料等级→ECT）
  - [ ] 湿度修正系数（50%/70%/90% RH）
  - [ ] 堆叠层数计算
  
- [ ] **Mullen耐破强度**
  - [ ] 材料等级→psi评级查表
  - [ ] 最大承重计算
  
- [ ] **堆叠安全系数**
  ```
  MaxLayers = BCT / (BoxWeight × SafetyFactor)
  ```
  - [ ] 安全系数表（仓储5.0/运输6.0/零售3.0）

#### 6. 缓冲保护算法
- [ ] **跌落高度→G值计算**
  ```
  G = sqrt(2 × h / d)
  ```
- [ ] **缓冲曲线查表**（泡沫类型×厚度×静应力→传递G值）
- [ ] **产品脆值数据库**（电子产品40-60G/玻璃制品25-40G）
- [ ] **内衬推荐引擎**（产品重量+脆值→泡沫类型+厚度）

#### 7. 印刷工艺约束规则引擎
- [ ] 最小线宽检查（胶印0.1mm/柔印0.15mm/数码0.08mm）
- [ ] 陷印规则（相邻色块0.1-0.2mm重叠）
- [ ] 套印精度（±0.3mm容差）
- [ ] 网点扩大补偿（Dot Gain）
- [ ] CMYK色域检查（超出Fogra39警告）
- [ ] 小字可读性（<6pt警告）

#### 8. 材料库与成本估算
- [ ] **材料库**（8种材料×厚度范围×单价）：
  - [ ] SBS漂白硫酸盐卡纸（0.3-0.8mm）
  - [ ] C1S/C2S卡纸（0.3-0.6mm）
  - [ ] 牛皮纸板（0.3-0.8mm）
  - [ ] 瓦楞纸E楞（1.5mm）
  - [ ] 瓦楞纸B楞（3.0mm）
  - [ ] 瓦楞纸BC双层（6.0mm）
  - [ ] 灰板/硬纸板（1.0-3.0mm）
  - [ ] 塑料PET/PP（0.2-0.5mm）
  
- [ ] **成本估算引擎**：
  - [ ] 材料成本（面积×单价×排版效率）
  - [ ] 印刷成本（方式×颜色数×覆盖面积）
  - [ ] 后加工成本（覆膜/烫金/UV/压印）
  - [ ] 模切成本（刀模费/批量分摊）
  - [ ] 组装/粘合成本
  - [ ] 阶梯价格（100/500/1K/5K/10K+）
  - [ ] 成本明细展示（柱状图/饼图）

#### 9. 材料承重与适配算法
- [ ] 产品重量→最低ECT要求
- [ ] 运输方式→最低Mullen要求
- [ ] 堆叠层数→最低BCT要求
- [ ] 材料降级推荐（满足强度前提下选最低成本）

#### 11. 智能包装推荐引擎（约束求解器）
- [ ] **用户输入（极简化）**：
  - [ ] 产品类别（下拉选择）
  - [ ] 产品尺寸（长×宽×高）
  - [ ] 产品重量
  - [ ] 预算上限
  - [ ] 用途（零售/电商/礼品/运输）
  
- [ ] **约束优化算法**：
  ```
  minimize: 总成本
  subject to:
    BCT ≥ 产品重量 × 堆叠层数 × 安全系数
    Mullen ≥ 运输方式对应等级
    传递G值 ≤ 产品脆值
    总成本 ≤ 预算上限
  ```
  
- [ ] **求解过程（四步流水线）**：
  1. 盒型筛选（根据用途+产品尺寸）
  2. 材料筛选（根据强度约束）
  3. 内衬计算（根据缓冲需求）
  4. 成本排序（输出3档方案）
  
- [ ] **输出三档方案**：
  - [ ] 成本优化方案（最便宜）
  - [ ] 最佳性价比方案（推荐）
  - [ ] 高端方案（品质优先）
  
- [ ] **决策透明化**（"为什么推荐这个"）：
  - [ ] 显示每个约束的计算过程
  - [ ] 显示材料降级逻辑
  - [ ] 显示成本构成
  
- [ ] **用户可手动"拧旋钮"**：
  - [ ] 调整安全系数（3.0-8.0）
  - [ ] 调整堆叠层数（1-10）
  - [ ] 锁定某个材料/盒型
  - [ ] 实时重算成本

4. **实时渲染** - PBR材质+折叠动画的性能优化
5. **成本估算准确性** - 需要真实供应商数据校准
