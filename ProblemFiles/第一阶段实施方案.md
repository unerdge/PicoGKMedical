## 第一阶段目标：完整的盒型结构引擎

### 实施步骤

#### Step 1: 扩展盒型类型（当前只有3种，需要10种）

**文件：** `3DBoxCanvasDesigner/BoxTypes/`

创建每种盒型的专用生成器：

1. **TuckEndBoxGenerator.cs** - 正插口盒
   - 参数：翻盖深度、插舌宽度、锁扣类型
   - 生成：6面体+顶部翻盖+底部翻盖+侧面插舌

2. **ReverseTuckEndGenerator.cs** - 反插口盒
   - 顶部和底部翻盖方向相反

3. **AutoLockBottomGenerator.cs** - 自动锁底盒
   - 底部自动锁扣结构（4个折叠翼）

4. **MailerBoxGenerator.cs** - 飞机盒/邮寄盒（当前已有基础版）
   - 增强：自锁扣设计、防尘翼

5. **CorrugatedRSCGenerator.cs** - 瓦楞RSC（当前已有基础版）
   - 增强：顶部和底部4片翼板

6. **PillowBoxGenerator.cs** - 枕头盒
   - 弧形边缘、两端折叠封口

7. **GableBoxGenerator.cs** - 提手盒
   - 顶部提手结构、锁扣

8. **RigidBoxGenerator.cs** - 天地盖精装盒
   - 两件式：盒盖+盒底（分离生成）

9. **SleeveTrayGenerator.cs** - 抽拉托盘盒
   - 外套筒+内托盘（两个独立Voxels）

10. **DisplayBoxGenerator.cs** - 展示盒
    - 前面开口、背面支撑板

#### Step 4: 刀版导出器

**文件：** `3DBoxCanvasDesigner/Export/`

1. **PdfExporter.cs** - 导出PDF刀版
   - 使用库：PdfSharp 或 iTextSharp
   - 包含：切割线（实线）、折叠线（虚线）、尺寸标注

2. **DxfExporter.cs** - 导出DXF（用于模切机）
   - 使用库：netDxf
   - 图层分离：CUT层、FOLD层、GLUE层

3. **SvgExporter.cs** - 导出SVG
   - 使用库：Svg (System.Drawing.Common)

#### Step 5: 结构验证器

**文件：** `3DBoxCanvasDesigner/Validation/StructureValidator.cs`

```csharp
public class StructureValidator
{
    public ValidationResult Validate(BoxParameters parameters)
    {
        var result = new ValidationResult();
        
        // 检查1：翻盖重叠是否充足
        if (parameters.FlapDepth < parameters.HeightMM * 0.5f)
        {
            result.AddWarning("翻盖深度不足，建议至少为高度的50%");
        }
        
        // 检查2：插舌比例是否合理
        if (parameters.TabWidth < 15f)
        {
            result.AddError("插舌宽度过小，最小15mm");
        }
        
        // 检查3：折叠半径是否考虑材料厚度
        float minRadius = parameters.WallThicknessMM * 1.5f;
        // ...
        
        return result;
    }
}

public class ValidationResult
{
    public List<string> Errors { get; set; }
    public List<string> Warnings { get; set; }
    public bool IsValid => Errors.Count == 0;
}
```

#### Step 6: 增强BoxParameters

**文件：** `3DBoxCanvasDesigner/BoxParameters.cs`（修改）

```csharp
public record BoxParameters(
    BoxType Type,
    float LengthMM,
    float WidthMM,
    float HeightMM,
    float WallThicknessMM = 2.0f,
    
    // 新增字段
    float FlapDepth = 0f,        // 翻盖深度（0表示自动计算）
    float TabWidth = 20f,        // 插舌宽度
    LockType LockType = LockType.Tuck,  // 锁扣类型
    bool AutoCalculateFlap = true       // 自动计算翻盖尺寸
);

public enum LockType
{
    Tuck,           // 插舌式
    AutoLock,       // 自动锁底
    Snap,           // 卡扣式
    Glue            // 粘合式
}
```
